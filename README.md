🚀 SmartCalc India

🔗 Live Website: https://calculator-kappa-inky-48.vercel.app/

SmartCalc India is a fast, simple, and free web application that provides multiple financial and utility calculators designed especially for users in India.

📌 Features
💰 EMI Calculator – Calculate loan EMI instantly
🧾 GST Calculator – Add or remove GST easily
📈 SIP Calculator – Estimate investment returns
🏷️ Discount Calculator – Calculate discounts quickly
🎓 Attendance Calculator – Track attendance and know how many classes to attend or bunk
🎯 Purpose

This project is built to:

Help students and individuals make quick calculations
Provide easy-to-use tools without login or complexity
Learn and implement real-world web development + SEO strategies
🛠️ Tech Stack
HTML
CSS
JavaScript
Hosted on Vercel
⚙️ How It Works

Each calculator is designed with a simple UI and instant computation logic:

EMI Calculator → Uses loan formula to compute monthly payments
GST Calculator → Adds/removes GST based on percentage
SIP Calculator → Estimates investment growth over time
Discount Calculator → Calculates final price after discount
Attendance Calculator → Predicts required attendance or bunkable classes
